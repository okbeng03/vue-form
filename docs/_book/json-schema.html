JSON Schema

JSON Schema是Gary Court 向 IETF 提交的用于描述 JSON 数据结构的规范； 其中包含配套工具可以满足 表单数据描述 和 表单校验 功能；同时一份通用描述可以统一 前端 & 后端 校验逻辑，保证体验一致性。

更多 JSON Schema 的描述、关键字定义等可以在此参考
