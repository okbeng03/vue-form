# Vue Form
使用Vue.js通过JSON schema自动生成表单。

## 它是什么？
Vue Form 是一组Vue components。它可以完成以下事情让表单页面的搭建变得容易：
1. 通过JSON schema生成表单
2. 通过JSON schema校验表单

## 浏览器支持
因为Vue.js只支持IE9+，所以我们也只能支持IE9+
